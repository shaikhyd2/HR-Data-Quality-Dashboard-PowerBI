# HR-Data-Quality-Dashboard-PowerBI
Power BI dashboard for analyzing HRBP data quality issues, including missing assignments, inactive mappings, and hierarchy validation.

# Project Overview

This project focuses on identifying and analyzing HR data quality issues related to HR Business Partner (HRBP) assignments.
The dashboard is built using Power BI to help organizations monitor inconsistencies such as missing HRBP mappings, incorrect hierarchy, and inactive assignments.

The goal is to provide clear visibility into data quality gaps and support better decision-making in HR operations.

# Key Insights and Features
1) Identifies Managers without assigned HRBP
2) Detects cases where HRBP is assigned to themselves
3) Highlights Active managers linked to inactive HRBP
4) Identifies incorrect HRBP hierarchy assignments
5) Identifies incorrect sub-family mappings
6) Visualizes Active vs Inactive Employees distribution
7) Shows HRBP status distribution (Active vs Inactive)
8) Interactive “View Detailed Data” navigation buttons
9) Clean and user-friendly dashboard design
